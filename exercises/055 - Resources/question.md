# Question 55: Define resource requests and limits

Create a `Deployment` named `resource-demo` with these requirements:

- Use image `nginx:1.27`
- Run `2` replicas
- Set CPU request to `100m`
- Set CPU limit to `200m`
- Set memory request to `128Mi`
- Set memory limit to `256Mi`

Save the manifest as `deployment.yaml` and apply it.
